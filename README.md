<p>Index_intellect is a web application that helps users generate a personalized study plan based on a book's table of contents and their learning goals. The application uses AI to analyze the book's content and generate a study plan that is tailored to the user's goals.</p>
<p>It is a single page application that is built using React and Vite. The application is deployed on Vercel and can be accessed at <a href="https://index-intellect.vercel.app">index-intellect.vercel.app</a></p>
